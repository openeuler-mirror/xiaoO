开发语言：Rust

什么是xiaoO？

xiaoO是AgentOS的智能中枢，让整个操作系统成为Agent的家

  • 自治系统管理 - 智能化的系统资源调度与运维决策
  • 无缝Agent编排 - 统一的Agent生命周期管理与协作
  • 开箱即用的智能能力 - 跨渠道的统一接入与丰富预置角色

核心价值

  • 让Agent拥有整个OS
    不再是Agent在系统中运行，而是整个系统成为Agent的家园
  • 统一的智能入口
    所有资源、服务、能力集中管理，一个中枢多种渠道
  • 高性能Rust实现
    内存安全、并发高效、模块化架构、易于扩展

快速开始

从源码安装：

git clone https://gitcode.com/openeuler/xiaoO.git
cd xiaoO
cargo build --release
cargo install --path apps/xiaoo-app

配置文件 (~/.config/xiaoo/config.toml)：

[llm]
provider = "openrouter"
model = "z-ai/glm-5"
api_key_env = "OPENROUTER_API_KEY"

使用方式

  • TUI模式（终端界面）
    xiaoo-tui
  • CLI模式（命令行）
    xiaoo run -p "Your Command"
  • Daemon模式（后台服务）
    xiaoo-app daemon --port 8080

Agent角色预设

可扩展的Agent角色系统：

[agent.code-reviewer]
description = "Reviews code for best practices"
prompt = "Focus on security, performance..."
tools.file_write = false
tools.file_edit = false

调用方式：

{ "text": "Review this patch", "agent": "code-reviewer" }
